# 实证论文中文精要撰写 Skill 使用指南

## 简介

这个 skill 专门用于将经济学、金融学、计量经济学等领域的英文实证论文转化为高质量的中文精要。它能够帮助你快速理解论文的核心内容、研究方法和主要结论，特别适合教学和学术交流。

## 主要功能

✅ 自动提取论文的研究问题、数据、方法和结论
✅ 详细解释 DID、PSM、SDID、RDD、IV 等计量方法
✅ 整理论文中的 Stata/R 代码（如果提供）
✅ 按照 myAPA 格式整理参考文献
✅ 生成结构化的 Markdown 文档

## 安装方法

1. 在 Claude.ai 界面中，点击右上角的设置图标
2. 选择 "Skills"（技能）
3. 点击 "Add skill"（添加技能）
4. 上传 `empirical-paper-summary.skill` 文件
5. 启用该技能

## 使用方法

### 基本用法

上传论文 PDF 文件，然后输入：

```
请帮我写这篇论文的中文精要
```

或者：

```
根据这篇论文写一份中文解读
```

### 提供代码文件

如果你有论文对应的 Stata/R 代码文件，可以一起上传：

```
请根据这篇论文和代码文件写中文精要，并在合适的位置插入关键代码
```

### 指定特殊要求

你也可以提出特殊要求：

```
请写中文精要，重点关注论文的识别策略和稳健性检验部分
```

## 输出内容

生成的中文精要包含以下部分：

1. **文档头部**
   - 标题：`中文精要：作者-年份-期刊`
   - 完整引文信息（myAPA 格式）
   - 关键词（包括方法、命令、主题等）

2. **正文章节**
   - 简介：研究背景、问题、发现、方法
   - 政策背景/研究问题：详细背景介绍
   - 数据来源与变量：数据、样本、变量定义
   - 研究设计/识别策略：模型设定、识别假设
   - 主要实证结果：基准回归、稳健性检验
   - 机制检验：中介效应、调节效应等
   - 结论与政策建议：总结、贡献、局限

3. **参考文献**
   - myAPA 格式的参考文献列表
   - 包含 DOI、PDF、Google Scholar 链接

## 格式特点

✨ **中英文混排规范**：英文和数字两侧加空格
✨ **标点符号正确**：使用中文引号「」和全角标点
✨ **数学公式美观**：使用 LaTeX 格式
✨ **代码格式规范**：60-70 列宽，Stata 使用 `///` 换行
✨ **学术风格严谨**：完整段落，论证清晰

## 代码整合示例

如果提供了代码文件，skill 会在适当位置插入代码：

```stata
// 基准 DID 回归
reghdfe EC policy Deve Population Land ///
    Structure Energy ER Income Industry, ///
    absorb(id year) vce(cluster id)
```

## 适用论文类型

- ✅ 政策评价类论文（如绿色金融政策、碳交易等）
- ✅ 实证检验类论文（如公司财务、劳动经济等）
- ✅ 方法论类论文（如新的 DID 方法、因果推断等）
- ✅ 使用 DID、PSM-DID、SDID、RDD、IV 等方法的论文
- ✅ 面板数据分析、空间计量分析等

## 参考文献格式说明

本 skill 使用 myAPA 格式整理参考文献，这是一种改进的 APA 格式，特点是：

- 优先使用 DOI 链接：`[Link](https://doi.org/xxx)`
- 提供 PDF 链接（如果可获取）：`[PDF](URL)`
- 必须包含 Google Scholar 链接：`[Google](<https://scholar.google.com/...>)`
- 不同类型文献（arXiv、NBER、期刊论文）格式略有不同

详细格式要求已内置在 skill 中，无需手动调整。

## 注意事项

1. **论文语言**：主要处理英文论文，输出中文精要
2. **论文格式**：支持 PDF、Word 等常见格式
3. **代码语言**：支持 Stata、R、Python 等
4. **输出长度**：通常 500-800 行，视论文复杂度调整
5. **分次输出**：如果内容很长，会分多次输出

## 示例场景

### 场景 1：课堂教学
教授可以快速为学生准备论文解读材料：
```
请把这篇 Arkhangelsky et al. (2021) 关于 SDID 的论文写成中文精要，
重点说明 SDID 方法与传统 DID 和合成控制法的区别
```

### 场景 2：文献综述
研究生可以快速理解领域内的重要论文：
```
请写这篇绿色金融政策评价论文的中文精要，我需要了解它的识别策略
```

### 场景 3：复现研究
研究者可以结合代码理解论文方法：
```
这是论文和对应的 Stata 代码，请写中文精要并标注关键代码
```

## 技术支持

2. 检查上传的文件格式是否支持
3. 如果输出不完整，可以要求继续输出剩余部分
